Technical Design

Hardware, firmware, and lighting behavior — one complete system.

Download Markdown

The build centers on an ESP32 with an RGB LED strip, PIR motion sensor, potentiometer, toggle switches, three photoresistors, and supporting passives mounted on perfboard inside a 3D printed shell. The LED strip uses a separate power rail to keep the ESP32 stable under full lighting load.

Firmware is written in C++ on the Arduino ESP32 toolchain. WiFiManager handles first-time WiFi setup, NTP provides time synchronization, and the embedded web server serves both the local interface and REST API endpoints.

  • Three photoresistors for redundant ambient light readings
  • PIR motion sensor via SPI for sleep detection
  • Separate LED power rail prevents controller resets
  • Heat-set inserts allow easy prototype disassembly

Risk Assessment

Risks are clear, prioritized, and tied to practical mitigations.

Download Word Document

The highest-risk issue — power instability from LED current draw — is resolved by separating the LED rail from the ESP32 supply. Thermal management uses venting, heatsinking, and software brightness limits to handle LED heat build-up.

Closed

Power instability, WiFi setup friction, and component availability have all been resolved in the current design iteration.

Mitigated

LED heating and ESP32 storage pressure are addressed with heatsinking, brightness limits, and efficient firmware storage usage.

Open

Motion sensing reliability and mobile app scope creep remain tracked. MVP is scoped to the lamp and local web UI only.

The test plan validates startup, networking, time sync, API behavior, timed light transitions, and configuration persistence — ensuring every critical behavior is verified before deployment.

API & Color Control

Device identity, current color state, color updates, and configuration updates for both sleep and wake cycle parameters via REST.

Sleep phase color transitions, wake-up light ramp timing, and out-of-cycle stability across schedule boundaries.

Persistence
